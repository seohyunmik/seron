개발을 위한 IT 지식

스마트폰에서 사용할 수 있는 앱/웹 서비스를 구현하기 위한 4가지의 방식

Mobile Web

모바일 화면에 맞게 구성한 웹

웹 개발 방식과 동일(html, css, Java 등)
웹 브라우저로 실행 (풀 브라우저 방식)

 Web App

구동방식이 웹처럼 보이게 한 앱

웹 개발 방식과 동일(html, css, Java 등)
웹 브라우저로 실행 (단일 페이지 방식)

 Hybrid App

웹 앱과 네이티브 앱의
기능을 결합하여 개발된 앱

모바일 웹앱을 모바일 운영체제로 패키징
스토어에서 설치하여 실행(PC접속 불가)

 Native App

모바일 기기에 최적화된 언어로 개발된 앱

각 운영체제에 최적화되어 있는 개발 도구로 제작
(And : Java / iOS : Swift)
스토어에서 설치하여 실행(PC접속 불가)
